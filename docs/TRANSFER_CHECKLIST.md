# Transfer Checklist

## Seller transfers

- GitHub repository or source ZIP
- Cloudflare Pages setup notes
- Release ZIP
- Buyer documentation
- Any included domain, if purchased and included in the sale

## Buyer connects

- Custom domain
- Email marketing account
- AI API provider
- AdSense or ad network
- Affiliate accounts
- Analytics
- Gumroad/payment links
- Social media accounts

## Support boundary

This asset is sold as a completed digital website asset. Buyer customization, paid services, and third-party account setup are buyer responsibilities unless separately agreed.
