# Start Here — KingdomShift Homeschool Hub

Welcome to KingdomShift Homeschool Hub.

This package includes the website source code, deployment instructions, buyer documentation, admin configuration guidance, monetization notes, and release handoff materials.

## First Steps

1. Install dependencies with `npm install`.
2. Build with `npm run build`.
3. Deploy to Cloudflare Pages.
4. Connect your custom domain.
5. Open `/admin-config` to record branding, AI, analytics, email, affiliate, and monetization settings.
6. Replace placeholder third-party account IDs with your own accounts.

## Seller Support Boundary

This is a website asset transfer. Buyer is responsible for connecting their own third-party accounts and customizing the platform after purchase.
